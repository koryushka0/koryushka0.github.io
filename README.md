# Охота и Рыбалка 

Всем привет! ( ´ ▽ ` )ﾉ Это мой учебный проект — полноценный фронтенд для интернет-магазина. 
Здесь всё кликабельно, всё работает и (надеюсь) ничего не ломается!

### Что тут есть?

*   Главная страница с каталогом и хитами продаж
*   Каталог с фильтрацией, сортировкой и "живым" поиском
*   Карточка товара с подробным описанием
*   Корзина и избранное (всё сохраняется)
*   Полностью адаптивная верстка для мобилок 

### Технологии

Сайт сделан на чистом **HTML**, **CSS** и **JavaScript**, без всяких навороченных фреймворков. Всё по-честному! (￣▽￣)/
