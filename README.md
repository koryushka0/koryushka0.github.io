# Охота и Рыбалка

Всем привет! Это мой учебный проект — полноценный интернет-магазин с фронтендом и бэкендом.
Здесь всё кликабельно, всё работает и (надеюсь) ничего не ломается! (￣▽￣)/

### Что тут есть?

*   **Полноценный фронтенд:** Главная страница, каталог с фильтрацией и "живым" поиском, карточки товаров, корзина и избранное.
*   **Динамический бэкенд:** Серверная часть на Python (Flask), которая принимает заказы с сайта.
*   **Интеграция с Telegram:** При оформлении заказа все данные мгновенно приходят менеджеру (мне!) в Telegram-бот.
*   **Полностью адаптивная верстка:** Все страницы отлично выглядят и работают на мобильных устройствах.

### Технологии

*   **Фронтенд:** чистый **HTML**, **CSS** и **JavaScript**, без всяких навороченных фреймворков. Всё по-честному!
*   **Бэкенд:** **Python** + **Flask**.
*   **Хостинг:** Фронтенд развернут на **GitHub Pages**, бэкенд — на **PythonAnywhere**.
